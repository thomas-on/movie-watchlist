### Movie Watchlist 

An application that allows user to search for movies from the IMDB database, and have the search returns the movie details. It also allows the user to generate a watchlist based on which movie(s) they have added. It consists of two pages - an index.html and a watchlist.html.

The index.html page is a search page where the user can search for movie title. Calls were then made to the IMDB API with the title searched for and the results returned were displayed on the page. A button to add movie to watchlist saves data to local storage.

The watchlist.html page list the items that have been added to the local storage. The remove form watchlist button is also provided to allow users to remove items from the watchlist.
